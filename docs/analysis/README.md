# Morpheus Corpus Analysis

This directory holds corpus-wide syntheses of the Morpheus knowledge base.

The primary artifact is **[corpus-overview.md](./corpus-overview.md)** — a quality
baseline, tag / method / benchmark / base-model / compute landscape, and a list of
Phase E gaps and follow-ups. The analysis is based on **177 / 291** Docling-converted
papers as of **2026-04-21T04:04:33Z** (the Docling batch is still running; rerun
`/tmp/analyze_corpus.py` after it completes to refresh the numbers).
